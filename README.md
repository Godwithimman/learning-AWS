## 📌 Project Overview

This repository documents a complete 30-day structured learning journey through AWS fundamentals. Each day focuses on one AWS service or concept, explained clearly with real-world context, hands-on labs, architecture diagrams, and exam prep questions.

Whether you are brand new to cloud computing or brushing up for certification, this repository is designed to take you from zero to confident in AWS fundamentals.

---

## 🎯 Learning Objectives

By the end of this 30-day journey, you will be able to:

- Explain what cloud computing is and why it matters
- Describe the AWS global infrastructure (Regions, Availability Zones, Edge Locations)
- Use and explain core AWS services: EC2, S3, RDS, Lambda, IAM, VPC, and more
- Understand AWS security, compliance, and the shared responsibility model
- Explain AWS pricing, billing, and cost management tools
- Apply the AWS Well-Architected Framework to cloud design decisions
- Prepare confidently for the AWS Certified Cloud Practitioner exam
- Deploy a simple real-world architecture on AWS

## 📚 Study Tips for the AWS Cloud Practitioner Exam

1. **Use the AWS Free Tier** — Hands-on practice is the best teacher. Sign up and experiment.
2. **Read the AWS Whitepapers** — The Well-Architected Framework and Shared Responsibility Model are exam favorites.
3. **Focus on the "Why"** — Understand why each service exists, not just what it does.
4. **Practice with sample questions** — Use AWS Skill Builder, Whizlabs, or Tutorials Dojo.
5. **Know the pricing model** — On-Demand vs Reserved vs Spot is commonly tested.
6. **Understand IAM deeply** — Users, groups, roles, and policies appear in many exam scenarios.
7. **Review the six pillars** of the Well-Architected Framework.
8. **Take timed practice exams** in the final week before your test.
