## 🃏 Exam Quick Reference

### Exam Details
- **Duration**: 90 minutes
- **Questions**: 65 (50 scored + 15 unscored)
- **Passing Score**: 700/1000 (70%)
- **Cost**: $100 USD
- **Format**: Multiple choice and multiple response
- **Delivery**: Online proctored or test center

### Domain Weightings
| Domain | Weight |
|---|---|
| Cloud Concepts | 24% |
| Security and Compliance | 30% |
| Cloud Technology and Services | 34% |
| Billing, Pricing and Support | 12% |

### Top Tested Topics
1. Shared Responsibility Model
2. IAM users, groups, roles, policies
3. EC2 pricing (On-Demand, Reserved, Spot)
4. S3 storage classes
5. AWS Support Plans
6. Well-Architected Framework 6 pillars
7. AWS Trusted Advisor
8. AWS Organizations and SCPs
9. VPC components (subnets, Security Groups, NACLs)
10. AWS Global Infrastructure (Regions, AZs, Edge Locations)

## 💡 Study Tips

1. **Start with the exam guide** — Download it from AWS and review every topic listed
2. **Hands-on practice** is the best teacher — use the AWS Free Tier
3. **Take full practice exams** under timed conditions at least 3 times
4. **Review every wrong answer** — understand why, not just what the right answer is
5. **Focus on the "why"** — Why does this service exist? What problem does it solve?
6. **Make flashcards** for pricing models, support plans, and the Well-Architected pillars
7. **Read AWS blog posts** about the services you struggle with
8. **Join AWS communities** — r/AWSCertifications on Reddit, AWS re:Post
