## How to Create Architecture Diagrams

### Recommended Free Tools

1. **draw.io / diagrams.net** — Free, browser-based, AWS icon library built-in
   - Go to https://diagrams.net
   - Search AWS icons in the shape panel
   
2. **AWS Architecture Icons** — Official AWS icon set
   - Download from https://aws.amazon.com/architecture/icons/
   
3. **Lucidchart** — Free tier available, great AWS templates
   - Go to https://lucidchart.com

4. **Cloudcraft** — Specialized for AWS diagrams, free tier available
   - Go to https://cloudcraft.co
